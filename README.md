# react-pomodoro-app

React SPA Pomodoro timer for focused work time

---

## Usage

_localhost:_

```sh
# to install dependencies:
$ npm install

# to start the application:
$ npm run start

# in browser, visit http://localhost:3000
# to STOP, Ctrl+C in Terminal
```
